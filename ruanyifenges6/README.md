## 题目
使用 React 实现 Button 和 Card 组件（具体可参考antd）
## 提交格式
在当前文件夹下以自己的名字或者代号进行命名自己的文件夹，可使用create-react-app来创建自己的项目

注意你的`.gitignore`文件，不要将`node_modules`文件夹提交

## 评分标准
评分主要从以下几个方面进行判断
1. api设计是否合理
2. 代码质量
  - 变量命令
  - 鲁棒性
  - 代码逻辑
3. 文档是否清晰易读
4. ui是否美观
